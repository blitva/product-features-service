## Development

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

```sh
npm install
```

Start the dev environment
```sh
npm run react-dev
npm run start-dev
```

Seed the database
```sh
npm run seeder
```

Run tests
```sh
npm db-tests
npm server-tests
npm front-end-tests
```

Open the browser
```
http://localhost:4000
```
